# The Hosted Collection

**Boutique short-term and mid-term rental management.**
Dayton, Ohio · Cabo San Lucas, Mexico

[thehostedcollection.com](https://thehostedcollection.com)

The Hosted Collection is a boutique property management company built on the belief that guests deserve more than a clean room — they deserve an experience. This repo documents the technical foundation behind the brand: booking infrastructure, website integrations, and the design system that holds it all together.

---

## What's in here

| File/Folder | What it covers |
|---|---|
| `booking-widget/` | Hostaway search widget setup and Squarespace embed |
| `squarespace/` | Custom code blocks for the Squarespace site |
| `brand/` | Colors, typography, and design guidelines |

---

## Stack

| Layer | Technology |
|---|---|
| Website | Squarespace |
| Booking engine | Hostaway |
| Reservation widget | Hostaway JS embed |
| Fonts | DM Sans (Google Fonts) |
